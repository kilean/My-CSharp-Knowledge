# How to add Projects to Solutions  
My Visual Studio C# Sample Code  


在方案的地方點擊滑鼠右鍵→加入

![img](https://pic.pimg.tw/kilean/1559832649-2215413998_n.png)

可以新增專案，也可以加入現有專案。

 

選擇新增專案

![img](https://pic.pimg.tw/kilean/1559867235-3693010499_n.png)

這裡有時候我會選擇Windows Form 應用程式或類別庫，

選錯也沒關係，可以從專案屬性修改輸出類型

![img](https://pic.pimg.tw/kilean/1559867771-4177161592_n.png?v=1559867771)

 

專案加入後記得在方案總管中，在專案名稱地方，點擊滑鼠右鍵，設定為啟始專案，這樣執行的時候才會是這個專案

![img](https://pic.pimg.tw/kilean/1559867872-1870257090_n.png)

 

如果剛剛選擇加入現有專案，

![img](https://pic.pimg.tw/kilean/1559868310-4232696237_n.png)


要注意的是，當你有多個方案時要小心，加入專案並不會將整個檔案複製到此專案，而是共有的，

移動路徑時要小心造成其他方案無法使用此專案，也要小心寫程式時只有某方案才能使用的狀況。
